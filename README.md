# calculator-html-css
Web Calculator
A sleek, functional calculator interface built with HTML5 and CSS3. This project focuses on clean UI design and grid-based button layouts.

🚀 Features
Full Operator Support: Includes Addition, Subtraction, Multiplication, Division, and Modulus.

Functional Controls: Dedicated buttons for "Clear" (C) and "Delete" (DEL).

Responsive Display: A top-mounted input field for real-time calculation viewing.

Modern Aesthetic: Organized in a clear, intuitive button-row structure.

🛠️ Technologies Used
HTML5: For the structural layout of the calculator body and input elements.

CSS3: For the styling, positioning, and visual feedback of buttons (linked via cal-styles.css).
